All about Mechanical Engineering.

We intend to develop the following minimum feature set in our application for Mechanical engineering applications, focusing on Process Equipment and Piping.

### Process Equipment (Intelligent 3D & 2D Symbols)
- Vertical and Horizontal Vessels
- Heat Exchangers
- Filters
- Pumps
- Air Coolers
- Compressors
- Embedded parameters with direct editing support

### Piping 3D
- Intelligent Piping Creation
- Spools, Valves, Elbows, T-joints, and Y-joints
- Instruments (Pressure Gauges, Temperature Gauges)

### Code Checking
- **ASME Code**: ASME Section VIII

### Simulation & Support
- Integration with load cases (Temperature, Pressure)
- Supports modeling (Fixed, Pinned, Roller)
